- MAMMP / XAMMP とさようならできるLAMP on Dockerコンテナです。
- composerが入っているのでLaravelも動きます（他WordPressも）。使用できるメーラーは「sendmail」。
- docker-compose.yml ファイルがある場所でターミナルを開き、以下のコマンドを叩く　　
 
　※  ポート番号　80　/　3000　をそれぞれ使用します
```
$ docker-compose up -d　
```
